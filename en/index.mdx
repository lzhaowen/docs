This site documents the currently available Neural4D legacy management APIs. Open the `old` tab to browse endpoint paths, parameters, response fields, and request examples.

<Note>
  The English interface uses [openapi.json](/openapi.json). The Chinese interface uses [openapi.zh.json](/openapi.zh.json), which translates summaries and field descriptions into Chinese while keeping paths and field names unchanged.
</Note>

## Included capabilities

- Text-to-3D model generation
- Image-to-3D model generation and chibi-style model generation
- Picture generation and video generation
- Generation result lookup and job progress lookup
- Model format conversion and download
- Portrait image detection and points balance queries

## Version tabs

- `old`: the current legacy management API set
- `v1`: reserved for future v1 documentation
- `v2`: reserved for future v2 documentation
