本文档整理了 Neural4D 旧版管理接口。你可以在这里查看接口路径、请求参数、返回字段和请求示例。

<Note>
  这套 API 统一使用 `Bearer token` 鉴权，接口基址为 `https://alb.neural4d.com:3000`。大多数生成接口会先返回 `uuid` 或 `uuids`，你需要再调用查询接口获取最终结果。
</Note>

## 接口范围

- 文本生成 3D 模型
- 图片生成 3D 模型和 Q 版模型
- 图片生成和视频生成
- 查询生成结果和任务进度
- 转换模型格式并下载结果
- 检测人像图片和查询剩余点数
